# 🩸 Blood Bank & Donation Management System

## 📌 Description

A web-based application to manage blood donors, blood availability, and requests. This system helps users find blood easily and allows admin to manage records.

---

## 🚀 Features

* Donor Registration
* Search Blood Groups
* Blood Request System
* Admin Dashboard
* Dynamic Pages (About Us, Why Donate, Contact)

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: Apache (XAMPP / AWS EC2)

---

## ⚙️ Installation (Local Setup)

1. Install XAMPP
2. Copy project to:
   C:\xampp\htdocs\
3. Start Apache & MySQL
4. Open phpMyAdmin
5. Create database:
   blood_donation
6. Import:
   /sql/blood_bank_database.sql
7. Open:
   http://localhost/BDMS

---

## ☁️ Cloud Deployment

Deployed using AWS EC2 (Ubuntu + Apache + MySQL)

---

## ⚠️ Note

Update database credentials in `conn.php` before running.
